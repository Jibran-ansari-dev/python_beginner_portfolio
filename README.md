# Python Beginner Portfolio

This portfolio showcases **7 Python projects** created to demonstrate foundational programming skills, file handling, data processing, and basic automation.  

Each project is organized in its own folder with a `main.py` script. Sample input files are included, and an `output/` folder is added where applicable.

---

## Projects Overview

### 1. Username Validator- **Folder:** `01_username_validator/`  
- **Files:** `main.py`, `sample_input.csv`  
- **Description:** Validates usernames based on length, allowed characters, and format rules.  
- **Skills:** Conditional logic, string methods, input validation.

### 2. Password Validator
- **Folder:** `02_password_validator/`  
- **Files:** `main.py`, `sample_input.csv`  
- **Description:** Checks passwords for length, uppercase/lowercase letters, digits, and special characters.  
- **Skills:** Boolean logic, string handling, input validation.

### 3. Email Validator
- **Folder:** `03_email_validator/`  
- **Files:** `main.py`, `sample_input.csv`  
- **Description:** Validates email addresses for correct structure and domain format.  
- **Skills:** String parsing, conditionals, input handling.

### 4. Combined Validator
- **Folder:** `04_combined_validator/`  
- **Files:** `main.py`, `sample_input.csv`  
- **Description:** Combines username, password, and email validation in a single CSV-based script.  
- **Skills:** File handling, CSV parsing, modular programming.

### 5. Text Analyzer
- **Folder:** `05_text_analyzer/`  
- **Files:** `main.py`, `sample.txt`, `output/`  
- **Description:** Reads a text file, counts lines, words, characters, sentences, shortest/longest words, and generates word frequency CSV reports.  
- **Skills:** File handling, string processing, dictionaries, sorting, CSV output.

### 6. Log Analyzer
- **Folder:** `06_log_analyzer/`  
- **Files:** `main.py`, `sample_log.txt`, `output/`  
- **Description:** Reads a timestamped log file, outputs chronological entries, counts actions and statuses, tracks first/last occurrences, and saves a CSV report.  
- **Skills:** File parsing, datetime handling, dictionary aggregation, CSV generation.

### 7. CSV Cleaner
- **Folder:** `07_csv_cleaner/`  
- **Files:** `main.py`, `input.csv`, `output/`  
- **Description:** Cleans CSV files containing usernames, passwords, and emails by removing spaces and lowercasing where appropriate.  
- **Skills:** File handling, CSV reading/writing, data cleaning, exception handling.
